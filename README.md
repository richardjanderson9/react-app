# React App

> A modern React application powered by Vite.

## Quick Start

```bash
npm install
npm start
```

## Scripts

- `npm start` – Start development server
- `npm test` – Run tests
- `npm run build` – Build for production

## Structure

```
src/        # Source code
public/     # Static files
```
